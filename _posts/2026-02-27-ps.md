    <li>如果偏序集\(L\)的每一个子集都有一个最大下界(下确界、交、meet、\(\land\))，则每一个子集都有一个最大上界(上确界、并、join、\(\lor\))，从而这个偏序集是一个完全格。证明如下：
    <ul>
        <li>空集\(\emptyset\)也有下确界。空集的下确界是\(L\)中的最大元（记为\(T\)），因为所有元素都是空集的下界，其中最大的那个就是最大元。因此\(L\)中存在最大元\(T\)</li>
        <li>对于任意子集\(S\subseteq L\),考虑\(S\)组成的所有上界构成的集合\(U=\{x\in L| \forall s\in S, s\leq x\}\),由于\(T\)是最大元，它一定大于等于所有元素，所以\(T\in U\),即\(U\)非空。根据条件，\(U\)作为\(L\)的子集，存在最大下界\(m=\bigwedge U\)，可以证明\(m\)就是\(S\)的上确界。因此每个子集有上确界也有下确界，是完全格</li>
    </ul>
    </li>
  </ul>

        <li>钻石格(Diamond Lattice)\(M_3\):<ul>
            <li>元素：\(\{0,a,b,c,1\}\)</li>
            <li>关系：\(0\)是最小元，\(1\)是最大元，\(a,b,c\)三者互不可比，且都介于\(0\)和\(1\)之间</li>
            <li>破坏分配律：取 \(a \land (b \lor c)\)。由于 \(b \lor c = 1\)，所以 \(a \land 1 = a\)。但 \((a \land b) \lor (a \land c) = 0 \lor 0 = 0\)。\(a \neq 0\)，所以分配律不成立。</li>
        </ul></li>
        <li>五角格(Pentagon Lattice)\(N_5\):<ul>
            <li>元素：\(\{0,a,b,c,1\}\)</li>
            <li>关系：\(0 < a < b < 1\)，且\(0 < c < 1\)，但\(c\)与\(a,b\)不可比</li>
            <li>取\(b\land (a \lor c)=b\land 1=b\)，但是\((b\land a)\lor (b\land c)=a\lor 0=a\)，所以分配律不成立</li>
        </ul></li>
